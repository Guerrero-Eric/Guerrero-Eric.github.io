JSON

JSON stands for JavaScript Object Notation. It is a syntax for soring and exchanging data.
JSON is essentially text, written in JavaScript object notation. It is a lightweight data interchange format that is self describing and easy to understand.
It also has the advantage of being language independant so it can be read and used by ANY programming language.
