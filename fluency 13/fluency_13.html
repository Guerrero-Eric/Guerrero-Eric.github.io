CSS Animation


Animation is process of making shape changes and creating motions with elements. @Keyframes Keyframes will control the intermediate animation steps in CSS3. The @keyframe rules is aplyed to the elements that receive the animations.


Example of @keyframe syntaxe:

                @keyframes example1 {
                from {background-color: blue;}
                to {background-color: yellow;}
                }
